# Family and Friends Cycle Spot — Ride the Future

A cinematic promotional experience (60–90s) built with Next.js, Tailwind CSS, and Framer Motion. Deployable on Vercel.

- Modern, clean, premium visuals
- Animated scene timeline, brand elements, and upbeat music
- One-click Deploy via `vercel` CLI

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build && npm start
```

## Deploy (production)

```bash
vercel deploy --prod --yes --token "$VERCEL_TOKEN" --name agentic-588acc3b
```
